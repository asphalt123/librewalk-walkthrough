# Main Quest: **Captain, Go Home!**

## Walkthrough
- Return to the ship and report to **ADA** to close the case.
- If you sided with **Helen** instead of **Ludovico** during **By Instinct**:
  - Briefly speak with **Sedrick Kincannon** and **Helen Halcyon** before departure.

## Key Information
- This step concludes the **Murder on Eridanos** DLC storyline.
- Final rewards and dialogue vary based on previous choices.
